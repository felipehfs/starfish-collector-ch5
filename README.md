# Starfish collector

A simple game made using Libgdx that tell about a turtle who wants 
to be the best world star collector.

![turle](assets/turtle-big.png)

## Features
- Collision detection
- Cutscenes
- Animation 
- UI e Dialogs
- Multiple screen
